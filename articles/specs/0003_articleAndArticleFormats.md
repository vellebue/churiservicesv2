# Article and Article Formats User Story

Create new entities to manage articles and article formats.

# Entities Description  

Article entity (to store on table ARTICLES):

    - Id: Long autogenerated article id from db sequence
    - ArticleId: String unique article identifier, mandatory and unique constrained, max length 10 characters
    - ArticleName: String article name, mandatory, max length 150 characters
    - formats: List of ArticleFormat entities

ArticleFormat entity (to store on table ARTICLE_FORMATS) and article must have several article formats attached, at least one configured as reference unit:

    - Id: Long autogenerated article format id from db sequence
    - Description: String format description, mandatory, max length 50 characters
    - ReferenceUnit: Boolean true if this format is the reference unit, false otherwise. This field is mandatory and must be true for at least one format and only one format.
    - EanCode: String ean code, optional, max length 20 characters
    - EanType: String ean type, optional, max length 10 characters
    - SaleUnit: Boolean that indicates if this format is enabled for purchase or not
    - ConversionFactor: BigDecimal that indicates the number of reference units that compose this format. This field is mandatory and must be greater than 0.
    - ArticleUnitId: String that references the article unit for this format, mandatory, use only the article units defined for ArticleUnit entity.
    
# Use Cases

- Create a new article with its formats.
- Update an existing article with its formats. Update image should replace previous existing image of an article and its formats.
- Delete an existing article with its formats by its Article Id (numeric long id).
- Query articles by Article Id (numeric long id).
- Query articles by a filter of:
  - ArticleId: String unique article identifier, optional, max length 10 characters. It may admit '*' as wildcard to match several characters for article id.
  - ArticleName: String article name, optional, max length 150 characters. It may admit '*' as wildcard to match several characters for article name.
