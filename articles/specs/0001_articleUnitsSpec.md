# Article Unit User Story

Generate the Use cases to manage article measure units as described below.
An article unit is composed of:

    - Article Unit symbol
    - Article Unit description (in English)
    - Article Unit key to point traslation into other languages.

The data base table to handle this information is:

    C_ARTICLE_UNITS

Create messages properties files to handle translations for languages: Spanish, English, French and German.

Use Cases:

    - Query an article unit by its symbol.
    - Query all article units
